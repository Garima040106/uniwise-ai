# Roadmap (Future Work)

> Note: This project is an early prototype. The roadmap below is a living,
> high-level list of planned enhancements. Items are intentionally date-free
> and intended to communicate direction rather than enforce deadlines.

## Short-term (next milestones)

- Real-time collaboration (WebSocket support) for shared study sessions
- Improve analytics dashboard (interactive charts, Plotly.js or similar)
- Polish accessibility (WCAG 2.1 AA baseline)
- Add basic i18n scaffolding (English-first, prepare for Spanish/Hindi)
- Stabilize tests and CI: 80% backend, 70% frontend coverage targets

## Medium-term

- Mobile app (React Native) MVP for reviewing flashcards and notifications
- OpenAI GPT-4 integration toggle (optional provider support)
- Rich notifications & calendar integration (study reminders)
- GitHub Issues → Milestones mapping for short/medium-term work

## Long-term

- Enterprise integrations (Salesforce, major LMS connectors)
- Multi-university analytics benchmarking & anonymized datasets
- Advanced personalization using longitudinal models (recommendation engine)
- Public developer docs + SDKs for third-party integrations

## How we use this file

- This file is a directional reference for contributors and interviews.
- For active task planning, please use GitHub Issues and Milestones.
- Keep descriptions high-level here; link to individual issues for specifics.

## Want to contribute?

If you'd like to help implement any item, open an issue or look for items
tagged `good-first-issue` and `roadmap`. Maintainers will triage and add
milestones as needed.
